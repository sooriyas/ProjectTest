=== About Jobile ===
Jobile is a lightweight, fast, clean and super flexible bootstrap based responsive theme for your job portal website. With this theme you can create your own careers / openings website for your business. Jobile theme can also be used to create classified ads, blog , business or personal websites also. Jobile has got theme options to manage general settings and also custom widget to manage social icons and text areas in the footer.

== Installation ==

1. Primary:
 = Login to your wp-admin account and go to Appearance -> Themes.
 = Select "Install" tab and click on the "Upload" link.
 = Select "jobile.zip" and click on "Install Now" button.
 = In case of errors, use an alternate method.
 
2. Alternate:
 = Unzip the template file (jobile.zip) that you have downloaded.
 = Upload the entire folder (jobile) to your server via FTP and place it in the /wp-content/themes/ folder.
 = Do not change the directory name.
 = The template files should be there now: /wp-content/themes/jobile/index.php (example).
 
3. Log into your WP admin panel and click on "Appearance". Go to "Themes" tab.

4. Now click on "jobile" to activate the theme.

5. Two custom widgets :

	i) Jobile social icon widget for display social icon in sidebar or footer.
	ii) Jobile image description widget for dispay image and description in sidebar or footer.
	
6. Theme Options :
	You can manage here site logo,favicon and footer copyright text. 

7. Google Maps:

	You can manage google map address by following.
	i)  Maps -> My first map [edit] -> Markers ->  Address/GPS:(Enter the address)  ->  Add Marker
	ii) Your Markers -> It will display below "Your Markers"
	iii) Added Google map will be visible to your main blog page.
